## 3. Graph Parser
**File:** `core/parser.py`  
- Reads graphs generated by `graph-generator.py`.  
- Converts JSON format into an **adjacency list**:
  ```python
  {0: [(2, 93)], 1: [(2, 24)], 2: [(0, 93), (1, 24)]}
  ```

- Handles **undirected and directed graphs**.  
- Raises custom `GraphParseError` on invalid files.
